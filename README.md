## 🎯 Overview

This port scanner is a lightweight yet powerful tool for identifying open TCP ports on a target host. It leverages Python's built-in `socket`, `threading`, and `sys` modules to deliver fast, concurrent scanning without external dependencies. The script is ideal for:

- Network security audits
- Infrastructure reconnaissance
- Educational demonstrations
- Cybersecurity portfolio projects

The scanner implements intelligent thread pooling, connection timeouts, and comprehensive error handling to ensure reliable operation in diverse network environments.
